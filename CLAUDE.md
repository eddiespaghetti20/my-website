# CLAUDE

This repository is a personal portfolio website built with plain HTML and CSS.

## Project Style
- Dark theme
- Modern design
- Clean layout
- Minimal dependencies

## Commit Message Guidelines
Use descriptive, focused commit messages. Keep each commit small and self-contained.

Suggested commit formats:
- `feat: add hero section`
- `fix: correct layout on mobile`
- `style: refine button hover states`
- `docs: add Claude guidance file`
- `chore: update colors and spacing`

## Notes for Contributors
- Keep changes small and isolated.
- Preserve the dark, modern aesthetic.
- Prefer semantic HTML and clean CSS.
- Avoid adding frameworks unless necessary.
